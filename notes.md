# Python Basics Notes

## Variables

Variables store values.

Example:
x = 10
name = "Vashu"

## Data Types

- int → whole numbers
- float → decimal numbers
- string → text
- bool → True / False

## Type Checking

type(x)

## Type Conversion

int("10")
str(10)
float(10)

## Key Points

- Python is dynamically typed
- Variable type depends on value

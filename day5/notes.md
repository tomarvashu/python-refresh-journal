# Dictionaries Notes

## What is a dictionary?
A dictionary stores data in key-value pairs.

```python
employee = {
    "name": "Vashu",
    "role": "Operations & Technology Lead"
}
```

## Why dictionaries are useful
Dictionaries are useful when data has labels.

Instead of remembering index positions, we can access values by keys.

```python
employee["name"]
```

## Common Operations
- access value using key
- add new key-value pair
- update existing value
- remove key
- loop through keys and values

## Useful Methods
- `keys()`
- `values()`
- `items()`
- `get()`
- `update()`

## Key Understanding
- lists are good for collections
- dictionaries are good for structured records
- real business data often looks like dictionaries

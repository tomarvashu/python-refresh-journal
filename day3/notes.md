# Control Flow Notes

## Conditional Statements
Use conditionals to make decisions.

Example:
if amount > 100:
    print("High value")

## Comparison Operators
- ==
- !=
- >
- <
- >=
- <=

## Logical Operators
- and
- or
- not

## Loops
Loops repeat code.

### For loop
Used when iterating through a sequence.

### While loop
Used when repeating until a condition changes

## Key Understanding
- if/elif/else controls logic
- for loop is useful for lists and ranges
- while loop needs careful condition control

# Data Structures Notes

## Lists
Lists are ordered, mutable collections.

Example:
numbers = [10, 20, 30]

Useful methods:
- append()
- insert()
- remove()
- pop()
- sort()

## Tuples
Tuples are ordered but immutable.

Example:
point = (10, 20)

## Sets
Sets store unique values and are unordered.

Example:
skills = {"python", "sql", "python"}

## Key Understanding
- Use lists when values may change
- Use tuples for fixed grouped values
- Use sets to remove duplicates
